🎯 MWS-Causal-Recommendation: Breaking the Filter Bubble

📌 Overview

This repository contains the implementation of a novel causal inference approach designed to mitigate Autonomy Bias (often referred to as the "Filter Bubble" effect) in Deep Learning-based Recommendation Systems.

Traditional recommender systems, while highly accurate, tend to trap users in narrow content loops by aggressively optimizing for historical engagement. This project introduces the Causal Autonomy NeuMF, a model that leverages causal intervention (Inverse Probability Scoring - IPS) during the training phase to actively reward content diversity and penalize over-reliance on dominant historical genres, thereby restoring user autonomy without significantly degrading predictive accuracy.

✨ Key Features & Technical Highlights

Causal Intervention via IPS: Implements dynamic sample weighting during training. Items falling inside a user's pre-calculated "causal filter bubble" are penalized, while items outside are rewarded, simulating a counterfactual world where the recommendation algorithm does not blindly follow historical bias.

Dual-Engine Architecture (Benchmarking):

MF Baseline: Standard Matrix Factorization.

NeuMF Baseline: Traditional Neural Collaborative Filtering (Neural Matrix Factorization).

Causal NeuMF: The proposed model incorporating causal intervention.

Comprehensive Evaluation Suite: Goes beyond standard accuracy metrics. Evaluates:

Accuracy: Recall@20, NDCG@20.

Autonomy/Diversity: Entropy scores.

Causal Impact: Average Causal Effect (ACE) and Paired T-tests for statistical significance.

Interactive Gradio Demo: Deployed on Hugging Face Spaces, allowing real-time, side-by-side comparison of recommendations generated by the Traditional NeuMF versus the Causal NeuMF for specific users.

📊 Dataset

The project utilizes the MovieLens 10M Dataset.

Processing: The pipeline extracts causal confounders by mapping each user's dominant genres based on highly rated movies (Rating $\ge$ 4.0) to define their individual "Filter Bubble".

📈 Academic Results & Benchmarks

Rigorous evaluation demonstrates that the proposed Causal NeuMF successfully increases user autonomy while maintaining highly competitive accuracy.

Average Causal Effect (ACE): Demonstrated a statistically significant positive shift in recommendation entropy (diversity).

Trade-off Analysis (Pareto Frontier): The sensitivity analysis confirms that moderate causal penalties yield a substantial reduction in Autonomy Bias with only a minimal drop in Recall@20.

(Note: Detailed performance charts including the Top-K Sensitivity and Accuracy vs. Autonomy Trade-off are generated during the training pipeline and uploaded to the HF repo).

🛠️ Tech Stack

Deep Learning Framework: PyTorch

Data Processing: Pandas, NumPy, SciPy (for Entropy calculations)

Deployment & UI: Gradio, Hugging Face Hub (huggingface_hub)

Architecture: Neural Collaborative Filtering (NCF / NeuMF)

🚀 Installation & Local Setup

Clone the repository:

git clone https://github.com/maherghanem86/MWS-Causal-Recommendation.git
cd MWS-Causal-Recommendation


Install dependencies:
Ensure you have Python 3.8+ installed.

pip install torch pandas numpy scipy tqdm huggingface_hub gradio


Run the Model Training Pipeline:
The entire pipeline (data downloading, bubble extraction, training, and evaluation) is contained within the primary Jupyter Notebook. Run the notebook to reproduce the models and charts locally.

Run the Gradio Interface Locally:
Execute the Gradio script to interact with the pre-trained models.

python app.py
